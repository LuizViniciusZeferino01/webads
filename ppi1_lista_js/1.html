O que é JavaScript ?

JavaScript foi criado em 1995 por Brendan Eich, em apenas 10 dias, para o Netscape Navigator. A ideia era adicionar interatividade às páginas da web, como validação de formulários, animações e outros recursos que rodassem no navegador, sem depender de servidores.

Navegador: JavaScript é interpretado pelo engine do navegador como V8 no chrome.

Servidor: Com plataformas como Node.js, JavaScript é utilizado para tarefas como acessar bancos de dados, processar requisições

O sandbox é um ambiente isolado que protege o sistema do usuário contra scripts maliciosos. Ele restringe o acesso do JavaScript à máquina local, permitindo interação apenas com o DOM e APIs controladas pelo navegador.

Document: Manipula a página (ex.: document.getElementById). Console: Exibe mensagens de depuração no console do navegador (ex.: console.log). Alert: Mostra uma caixa de diálogo com mensagem (ex.: alert('Olá!')).
