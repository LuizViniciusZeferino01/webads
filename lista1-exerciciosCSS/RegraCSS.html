Oque é uma regra CSS ?

Uma regra CSS é um código que diz como os elementos HTML devem aparecer na página. Ela define quais partes da página vão ter estilo e o que vai mudar nelas, como a cor, o tamanho ou o formato.


Exemplo prático utilizando seletor(h1), propriedade(color, font-size) e valor(green, 12px):

h1{color: green; font-size: 12px;}
